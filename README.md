# AWS-LangChain-LangGraph

# RAG System for AWS Notes (Version 1)

This project is a **Retrieval-Augmented Generation (RAG) system** for **AWS Notes** from the AI Practitioner Udemy course.  
It is built using **LangChain** and **LangGraph** and integrates additional tools like **WikipediaAPIWrapper** and **ArxivAPIWrapper** for enhanced information retrieval.

---

## **Project Overview**

- Ingested **AWS Notes** content to create a retrievable knowledge base.
- Uses **FAISS** as a vector store for fast semantic search.
- Implements a **QA chain** using **LangChain** and visual chaining with **LangGraph**.
- Integrates **Wikipedia** and **Arxiv** for additional context and extra knowledge.
- Built with **OpenAI GPT LLMs** for natural language responses.
- This is **version 1**, demonstrating the basic pipeline and multi-source retrieval.

---

## **Features**

- **PDF / Document ingestion** – Add AWS course notes.
- **Vector store retrieval** – Semantic search with FAISS.
- **QA chain** – Ask questions in natural language.
- **Extra tools** – Wikipedia & Arxiv API wrappers for supplementary information.
- **LangGraph visual chaining** – Easy to see and modify the QA pipeline.
